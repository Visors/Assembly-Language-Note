# 3.4 汇编器清单文件

建议自行运行，得到.LST文件，打开对照.asm看一看。之前说过的3.2说过的output模式在这里能看的很清楚。

## .DATA

### 形式

偏移量 内存内容 名称 类型 语法内容

### 偏移量

偏移量从00000000开始，根据DATA声明，各语句有对应的偏移量：

| 单位  | 产生偏移量/数据 |
| :---: | :-------------: |
| BYTE  |        1        |
| WORD  |        2        |
| DWORD |        4        |

### 内存内容

特别注意的是用DUP开出来的空间的内容，显示上是[00]，但并不是指空字符。
